# 移动端与平板端验收清单

## 设备档位

- [ ] 手机竖屏（<= 767px）
- [ ] 平板竖屏（768px - 1024px）
- [ ] 平板横屏（1025px - 1279px）
- [ ] 桌面（>= 1280px）

## 核心翻译主路径

- [ ] 进入书籍详情页后，可在当前档位完成“目录 -> 章节 -> 段落翻译”闭环
- [ ] 段落操作入口在触控设备可见（校对、润色、重译、复制到助手）
- [ ] 章节排序在触控设备可用（上移/下移），并与桌面拖拽结果一致
- [ ] 翻译进度面板可打开/关闭，切换后仍保留当前章节上下文
- [ ] 术语/角色/记忆设置在小屏可达，返回内容区后定位不丢失

## 壳层与弹层

- [ ] 手机/平板壳层切换不影响路由导航
- [ ] 桌面三栏交互保持原有行为（侧栏、主区、右侧助手）
- [ ] 多层弹层按后进先出关闭（Esc/关闭按钮）
- [ ] 编辑类 Dialog 在有未保存修改时，关闭前出现确认

## 次级页面

- [ ] `BooksPage` 在手机/平板可完成新增、编辑、删除、进入详情
- [ ] `AIPage` 在手机/平板可完成新增、编辑、测试、删除模型
- [ ] `HelpPage` 在手机支持文档抽屉与目录抽屉导航

## 回归与构建

- [ ] `bun test` 通过关键回归用例
- [ ] `bun run type-check` 通过
- [ ] `bun run lint` 通过
