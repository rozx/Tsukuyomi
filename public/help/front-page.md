# Tsukuyomi (月咏) 翻译器使用指南

欢迎使用 **Tsukuyomi (月咏) 翻译器**！这是一款专为日本轻小说翻译爱好者打造的现代化 AI 辅助翻译工具。

无论您是想要快速阅读生肉的读者，还是追求信达雅的专业翻译，Tsukuyomi 都能通过强大的 AI 模型（如 **GPT-5.1**, **Claude 4**, **Gemini 3.0 Pro**) 为您提供得力的支持。

---

## 🚀 快速开始 (Quick Start)

仅需三步，开启您的翻译之旅：

### 1. ⚙️ 配置 AI 模型

Tsukuyomi 是一个 "Bring Your Own Key" (自带密钥) 的应用。您需要配置 AI 服务商的 API Key 才能开始翻译。

1. 点击左边栏的 **AI 列表 (AI List)**。
2. 添加您拥有的 API Key：
   - **OpenAI**: 支持 **GPT-5.1**, **GPT-5**, **o1-pro** 等。
   - **Google Gemini**: 支持 **Gemini 3.0 Pro**, **Gemini 2.0 Flash**。
   - **Anthropic**: 支持 **Claude 4 Opus/Sonnet**, **Claude 3.5 Sonnet**。
   - **DeepSeek**: 支持 **DeepSeek-V3**, **DeepSeek-R1**。
   - **Moonshot (Kimi)**: 支持 **Kimi-k2.5**, **Kimi-v1.5**。
3. 点击 **保存** 并测试连接。

### 2. 📚 创建书籍

1. 点击 **书籍管理** 页面。
2. 点击右上角的 **新建书籍** 按钮。
3. 您有两种方式创建：
   - **手动创建**: 输入书名、作者、简介。
   - **自动抓取**: 粘贴小说源站链接（如 `syosetu.org`, `kakuyomu.jp`），系统会自动抓取元数据。

### 3. 📝 开始翻译

1. 进入书籍详情页。
2. 点击 **添加章节** 或 **批量抓取**。
3. 进入章节页面，点击右上角的 **开始翻译 (Start Translation)**。
4. 坐下来，看着 AI 为您实时生成流畅的译文。

## ✨ 核心功能详解

### 📖 翻译与阅读

- **沉浸式阅读模式**: 左侧原文，右侧译文，支持段落对照。
- **多版本对比**: 对不满意的段落，可以要求 AI "重译"，并对比不同版本的差异。
- **人工润色**: 双击任意译文段落即可进入编辑模式，您的修改将被保存并用于后续参考。

### 🧩 术语表 (Glossary)与一致性

这是 Tsukuyomi 最强大的功能之一，用于确保长篇小说翻译的一致性。

- **术语表 (Glossary)**:
  - 记录地名、道具名、必杀技等专有名词。
  - AI 在翻译时会严格遵守这些术语的译法。
- **角色设定 (Role/Character)**:
  - 定义角色的说话风格、性格特征、口癖（如“desu/masu”语气）。
  - AI 会根据对话的发言者（如果能识别）调整语语气。

### 🤖 翻译任务类型

除了基础翻译，我们还提供多种 AI 辅助任务：

- **初翻 (Translation)**: 将原文转换为中文。
- **润色 (Polishing)**: 优化已有的译文，使其更通顺、更符合中文表达习惯。
- **校对 (Proofreading)**: 检查译文中的错别字、漏译或语义错误。
- **摘要 (Summary)**: 自动生成章节摘要，帮助您快速回顾剧情。

### 💬 聊天助手会话 (Chat Assistant)

Tsukuyomi 提供了智能聊天助手功能,让您可以随时与 AI 进行深度交流:

- **翻译咨询**: 对某个段落、词汇或表达有疑问?直接向 AI 提问,获得详细的翻译建议和多种译法选择。
- **术语解释**: 不确定某个术语的最佳翻译?AI 会结合上下文和日本文化背景,为您提供专业的解释和建议。
- **风格探讨**: 询问不同翻译风格的优劣,比如"直译 vs 意译"、"正式 vs 口语化",AI 会帮您权衡利弊。
- **上下文记忆**: 每个书籍都有独立的聊天会话,AI 会记住您之前的对话内容,保持讨论的连贯性。
- **实时协作**: 在翻译过程中遇到难题时,可以立即打开聊天窗口,与 AI 并肩作战,共同攻克翻译难关。
- **使用帮助**: 询问如何使用 Tsukuyomi,AI 会为您提供详细的使用说明。

**使用提示**: 点击右上角的 **聊天助手** 图标即可开启对话。每本书籍的聊天记录会自动保存,方便您随时回顾讨论内容。

### 💾 数据同步与安全

- **本地优先**: 所有数据默认存储在您浏览器的 IndexedDB 中，并在本地加密，保护您的隐私。
- **Github Gist 同步**:
  - 在设置中配置 Github Token。
  - 利用 Gist 实现多设备间的无缝同步（如从电脑同步到手机继续阅读）。

## 💡 最佳实践 (Best Practices)

1. **AI 自动化数据维护**: 在翻译过程中，AI 会**自动提取并建议**新的术语、角色设定和记忆。您无需手动录入，只需在翻译完成后，进入相应的管理面板（Terminology/Characters/Memory）进行**核对与修正**。
2. **章节摘要的力量**: 翻译任务完成后，系统会自动生成 **章节摘要 (Chapter Summary)**。这些摘要是 AI 理解剧情走向、在翻译后续章节时保持逻辑连贯的关键，建议定期检查摘要的准确性。
3. **模型协作策略**:
   - **大模型翻译**: 推荐使用大型推理/思考模型（如 **DeepSeek-R1**, **GPT-o1**, **Claude 4 Opus**）进行初始翻译，它们能更好地处理复杂的长难句和语境。
   - **多阶段处理**: 在初翻完成后，您可以尝试切换至 **Claude 3.5 Sonnet** 或 **Gemini 2.0 Flash** 进行高速的 **润色 (Polishing)** 或 **校对 (Proofreading)**，不同模型的视角和语言风格组合通常能带来更自然的译文。

## ❓ 常见问题 (FAQ)

**Q: 为什么翻译有时候会中断？**
A: 这通常是由于 API 网络波动或额度耗尽。请检查您的 API Key 余额或网络连接（国内用户可能需要检查网络环境）。

**Q: 自动抓取支持哪些网站？**
A: 目前完美支持 `Shousetsuka ni Narou (成为小说家吧)`, `Kakuyomu`, `Hameln` 等主流日轻网站。

**Q: 如何反馈 Bug 或建议？**
