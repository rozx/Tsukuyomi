# Tsukuyomi 翻译器使用指南

欢迎来到 **Tsukuyomi (月咏) 翻译器**！这是一款专为日本轻小说翻译设计的 AI 驱动工具，旨在提供流畅、高效的翻译体验。

> **💡 提示**
> Tsukuyomi 支持 **OpenAI** 和 **Google Gemini** 等多种先进模型，为您提供最佳的翻译质量。

## ✨ 核心功能

### 📚 书籍管理

轻松管理您的翻译项目：

- **创建书籍**: 快捷导入或创建新的翻译项目
- **分卷分章**: 支持多卷多章的结构化管理，即使是长篇巨著也能井井有条
- **段落编辑**: 专业的逐段翻译界面，支持快速校对和润色

### 🤖 AI 翻译引擎

- **多模型支持**: 内置 OpenAI GPT-4o, Google Gemini Pro 1.5 等最新模型支持
- **流式响应**: 实时查看翻译进度，无需漫长等待
- **多版本对比**: 对同一段落生成的不同翻译版本进行对比，择优而录

### 📝 术语与记忆

确保翻译的一致性与准确性：

- **术语表 (Glossary)**: 统一管理专有名词、人名、地名
- **角色设定**: 记录角色语气、性格，让 AI 翻译更具代入感
- **AI 智能记忆**: 系统会自动记忆上下文，确保剧情连贯

### 🌐 一键抓取

支持从主流小说网站自动抓取内容：

- `syosetu.org` (Hameln)
- `kakuyomu.jp`
- `ncode.syosetu.com` (Syosetsuka ni Naro)
- `novel18.syosetu.com` (Nocturne Novels)

### 💾 数据同步与导出

- **云端同步**: 通过 GitHub Gist 实现多设备无缝同步
- **本地优先**: 使用 IndexedDB 本地持久化存储，离线也能工作
- **灵活导出**: 支持导出为 TXT, Markdown, HTML 等格式

## 🚀 快速入门

### 1. 配置 AI 模型

进入 **设置** 页面，添加您的 OpenAI 或 Google Gemini API 密钥。这是使用 AI 翻译功能的前提。

### 2. 创建第一本书

点击首页的 **新建书籍** 按钮。您可以输入书籍标题、作者，或者直接粘贴源站 URL 自动获取信息。

### 3. 添加章节

在书籍详情页，点击 **添加章节**。您可以：

- **手动输入**: 粘贴原文内容
- **自动抓取**: 输入章节 URL 自动获取

### 4. 开始翻译

进入章节页面，点击右上角的 **开始翻译** 按钮。AI 将自动分析并逐段翻译内容。

> **🌟以此为荣**
> 您随时可以点击任意段落进行人工润色，您的修改与 AI 的辅助将共同成就完美的译作。

### 5. 导出成品

翻译完成后，在书籍菜单中选择导出，即可获得排版精美的译文文件。
