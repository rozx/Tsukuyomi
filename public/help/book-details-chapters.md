# 📚 章节管理

本页说明书籍详情页中「卷 / 章节」相关的实际功能与行为。

## 📖 基本结构

- **卷（Volume）**：用于组织章节。
- **章节（Chapter）**：必须隶属于某个卷。

左侧「目录」区域支持：新建、编辑、删除、折叠/展开、拖拽调整章节顺序。

## ➕ 新建卷

1. 在左侧「目录」点击 **新卷**。
2. 输入卷标题。
3. 点击添加。

当前实现行为：

- 新卷会追加到卷列表末尾。
- 新建卷时仅填写标题；卷译名可在后续「编辑卷」中设置。

## ➕ 新建章节

1. 点击 **新章节**。
2. 在弹窗中选择所属卷并填写章节标题。
3. 点击添加。

当前实现行为：

- 若当前书没有任何卷，无法新建章节，会提示先创建卷。
- 新建章节会追加到所选卷末尾。
- 新建弹窗仅包含「所属卷 + 章节标题」，不在此处录入章节正文或章节译名。

## ✏️ 编辑卷

在卷标题右侧点击编辑按钮，可修改：

- 卷标题（原文）
- 卷译名（可选）

显示时会按当前显示规则渲染卷标题（是否显示译名取决于当前标题数据）。

## ✏️ 编辑章节

在章节右侧点击编辑按钮，可修改：

- 章节标题（原文）
- 章节译名（可选）
- 所属卷（可将章节移动到其他卷）
- 章节级特殊指令：翻译 / 润色 / 校对

说明：

- 通过编辑弹窗修改所属卷时，章节会被移动到目标卷末尾。
- 章节级特殊指令会覆盖书籍级同类指令，仅作用于该章节。

## 🗑️ 删除卷

在卷右侧点击删除按钮并确认后删除。

当前实现行为：

- 删除卷时会同时删除该卷下全部章节。
- 删除操作不可撤销（确认弹窗会明确提示）。

## 🗑️ 删除章节

在章节右侧点击删除按钮并确认后删除。

当前实现行为：

- 章节从所属卷中移除。
- 删除操作不可撤销。

## 🔄 拖拽章节排序

章节支持拖拽：

- **同卷重排**：调整章节在卷内顺序。
- **跨卷移动**：拖拽到其他卷中的目标位置。

拖拽反馈：

- 被拖拽章节会半透明。
- 目标章节或目标卷列表会显示拖拽高亮样式。

## 📂 卷折叠 / 展开

- 点击卷标题左侧箭头可折叠或展开该卷章节列表。
- 系统会按书籍维度记住卷展开状态（保存在本地 UI 状态中）。

## 👀 列表提示

- 当前选中章节会高亮。
- 章节悬停时显示编辑/删除按钮。
- 若章节存在摘要，标题右侧显示信息图标，悬停可查看摘要预览。

## ❓ 常见问题

### 为什么点了「新章节」却不能添加？

当前书还没有卷。请先创建至少一个卷。

### 删除卷会怎样？

会连同该卷下所有章节一起删除，且无法撤销。

### 能拖拽卷顺序吗？

当前实现仅支持「章节拖拽」，不支持卷拖拽排序。
