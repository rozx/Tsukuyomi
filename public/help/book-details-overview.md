# 📖 书籍详情页概览

书籍详情页是单本书的核心工作区，采用「左侧目录 + 右侧内容」布局。

---

## 🧭 页面结构

### 左侧区域（书籍与目录）

- 书籍卡片：封面、标题、卷数/章节数/字数统计（点击可编辑书籍信息）。
- 设置入口：
  - 术语设置
  - 角色设置
  - 记忆管理
  - 检查更新（从在线源合并章节）
- 目录区：
  - 新卷 / 新章节
  - 卷与章节列表
  - 卷展开/折叠
  - 章节拖拽排序（可跨卷移动）
- 底部返回：返回书籍列表。

### 右侧区域（当前工作区）

- 章节工具栏（撤销/重做、模式切换、搜索、导出、翻译相关操作等）。
- 搜索工具栏（查找/替换，按匹配项跳转）。
- 主体内容按状态切换：
  - 术语面板 / 角色面板 / 记忆面板
  - 章节内容编辑区
  - 可选翻译进度面板

---

## ✨ 核心能力

### 1) 章节结构管理

- 添加、编辑、删除卷与章节。
- 章节可拖拽调整顺序，并支持跨卷放置。
- 章节支持标题翻译、章节级特殊指令等信息维护。

### 2) 三种编辑模式

- **原始文本模式**：编辑章节原始文本。
- **翻译模式**：段落级原文/译文操作（默认工作模式）。
- **预览模式**：面向阅读的译文预览，未翻译段落会显示提示。

### 3) 段落与章节操作

- 段落可重译、润色、校对。
- 段落支持多译文版本切换。
- 章节可生成/重新生成摘要。
- 可查看本章术语、角色、记忆引用计数并打开对应弹出面板。

### 4) 搜索与快捷键

- 搜索工具栏支持查找、替换、全部替换。
- 常用快捷键（当前页面）：
  - `Ctrl+F`：打开/关闭查找
  - `Ctrl+H`：打开/关闭替换
  - `F3` / `Shift+F3`：下一个/上一个匹配
  - `Esc`：关闭搜索工具栏
  - `Ctrl+Z` / `Ctrl+Y`：撤销/重做
  - `Ctrl+Shift+C`：复制本章已翻译文本

### 5) 导出能力

章节导出菜单支持：

- 原文 / 译文 / 双语
- 复制到剪贴板 / 导出 JSON / 导出 TXT

---

## 🚀 基本使用流程

1. 在左侧目录选择章节。
2. 在右侧工具栏选择编辑模式（通常先用翻译模式）。
3. 进行段落翻译与修订，必要时使用搜索替换。
4. 通过术语/角色/记忆入口复核一致性。
5. 使用导出菜单导出章节成果。

---

## ℹ️ 与顶部工具栏的关系

在书籍详情页顶部还会显示“批量摘要”入口（全局工具栏中的条件按钮），可用于整本书章节摘要批处理；本页文档聚焦书籍详情区本身。
