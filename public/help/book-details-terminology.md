# 术语管理

术语管理功能帮助您维护专有名词、人名、地名等的翻译一致性，确保整本书的翻译质量。

## 📚 什么是术语库

术语库是书籍特定的专有名词翻译对照表，包括：

**常见术语类型**：

- 👤 **人名**: 角色名称的翻译
- 🗺️ **地名**: 地点、国家、城市等
- 📦 **专有名词**: 组织名称、技能名称、物品名称等
- 💡 **特殊概念**: 世界观相关的特定术语
- 🎌 **文化词汇**: 特定文化背景的词汇

## 📋 术语的结构

每个术语包含以下信息：

- 📝 **名称**: 原文中的术语（必填）
- 🔄 **翻译**: 对应的中文翻译
- 📄 **描述**: 额外的说明信息（可选）

## 🚪 访问术语管理面板

**操作步骤**：

1. 📑 在书籍详情页的左侧边栏
2. ⚙️ 找到"设置菜单"区域
3. 🖱️ 点击"术语设置"按钮
4. 📋 右侧会显示术语管理面板

**面板布局**：

- 🛠️ **顶部工具栏**: 添加术语、批量操作、导入/导出、搜索

- 📇 **术语列表**: 显示所有术语的卡片
- 📄 **术语详情**: 点击术语查看和编辑详细信息

## 添加术语

### 手动添加

**操作步骤**：

1. 在术语管理面板
2. 点击"添加术语"按钮
3. 在弹出的对话框中填写：
   - **名称**: 原文中的术语（必填）
   - **翻译**: 对应的翻译
   - **描述**: 额外的说明信息（可选）
4. 点击"保存"按钮

**填写建议**：

- **名称准确**: 必须与原文完全一致
- **翻译统一**: 使用规范的翻译
- **描述详细**: 添加必要的说明，如读音、含义等

### 使用翻译功能

在术语编辑对话框中：

- 名称输入框右侧有翻译按钮
- 点击可以自动翻译术语
- 翻译结果会填入"翻译"字段

## 编辑术语

### 修改术语信息

**操作步骤**：

1. 在术语列表中找到要编辑的术语
2. 点击术语卡片
3. 在弹出的对话框中修改信息
4. 点击"保存"按钮

### 修改影响

**术语修改后**：

- 修改不会自动更新已翻译的内容
- 新的翻译会使用更新后的术语
- 建议使用搜索替换功能更新旧译文

**更新已翻译内容**：

1. 修改术语后
2. 使用章节的搜索替换功能
3. 搜索旧译文，替换为新译文
4. 或者重新翻译相关章节

## 删除术语

**操作步骤**：

1. 在术语列表中找到要删除的术语
2. 点击术语卡片旁的"删除"按钮
3. 在确认对话框中确认删除

**删除影响**：

- 删除后不可恢复
- 已翻译的内容不会自动更改
- AI 翻译时不再使用该术语

## 批量操作

### 批量删除

**操作步骤**：

1.  点击工具栏的"批量"按钮
2.  勾选要删除的术语
3.  点击"删除"按钮
4.  在确认对话框中确认

## 导入与导出

### 导出术语

- 点击工具栏的"导出"按钮
- 将当前术语库导出为 JSON 文件
- 用于备份或迁移

### 导入术语

- 点击工具栏的"导入"按钮
- 选择 JSON 格式的术语文件
- 系统会自动合并新旧术语

## 术语搜索

### 快速查找术语

**操作步骤**：

1. 在术语面板顶部找到搜索框
2. 输入搜索关键词
3. 列表会实时过滤显示匹配的术语

**搜索范围**：

- 名称
- 翻译
- 描述

**搜索技巧**：

- 支持模糊搜索
- 不区分大小写
- 可以搜索部分关键词

## 术语在翻译中的应用

### AI 自动应用

**翻译时**：

- AI 会自动检测原文中的术语
- 使用术语库中的译文替换
- 保持翻译的一致性

**应用优先级**：

1. 完全匹配优先
2. 长术语优先（避免冲突）
3. 最近添加的术语优先

### 术语的作用

**名称翻译**：

- 统一使用设定的译名
- 确保同一名词翻译一致
- 避免翻译过程中的混乱

**上下文理解**：

- 描述字段帮助 AI 理解术语含义
- 更准确地处理术语相关内容

## 最佳实践

### 建立术语库的时机

**推荐流程**：

1. **导入书籍后立即建立**: 先创建基本的术语库
2. **翻译前完善**: 在开始翻译第一章前，尽可能完善术语库
3. **翻译中补充**: 翻译过程中发现新术语及时添加
4. **定期审核**: 定期检查和更新术语库

### 术语命名规范

**统一风格**：

- 人名统一使用"姓·名"或"名 姓"格式
- 地名统一使用官方译名或音译
- 技能名称统一使用"XX术"或"XX之XX"格式

**避免冲突**：

- 一个原文只对应一个译文
- 不同原文不要使用相同译文
- 使用描述区分相似术语

### 术语的优先级

**重要性排序**：

1. **核心角色**: 主角、重要配角的名字
2. **关键地点**: 主要场景、重要地标
3. **核心概念**: 世界观相关的重要术语
4. **常用技能**: 频繁出现的技能或魔法
5. **其他术语**: 次要角色、一般物品等

**建议**：

- 先完善高优先级的术语
- 低优先级术语可以在翻译中逐步添加

### 术语质量控制

**定期审核**：

- 每完成一卷后审核术语库
- 检查术语的一致性
- 修正错误或不当的翻译

### 术语与角色设定的区别

**术语库**：

- 存储所有专有名词
- 只包含名称和翻译
- 可选的描述信息

**角色设定**：

- 专门管理人物信息
- 包含性别、说话口吻等详细信息
- 支持别名管理

**建议**：

- 所有专有名词都添加到术语库
- 重要角色同时添加到角色设定
- 角色设定提供更丰富的上下文

## 常见问题

### 为什么 AI 翻译时没有使用术语库？

1. 检查术语的原文是否完全匹配
2. 确认术语库已保存
3. 尝试重新翻译该段落
4. 检查 AI 模型是否支持术语库

### 如何处理一词多义的术语？

1. 在描述中说明使用场景
2. 考虑使用章节特殊指令区分
3. 人工翻译时根据上下文选择

### 术语太多会影响性能吗？

- 一般不会影响性能
- 系统会智能缓存常用术语
- 建议控制在 1000 个术语以内
- 定期清理不再使用的术语

### 如何批量修改术语？

目前支持**批量删除**。如果需要批量修改内容：

1.  点击"导出"按钮备份数据
2.  修改导出的 JSON 文件
3.  重新"导入"并覆盖

### 翻译字段应该留空还是填写？

- **留空**: AI 翻译时会自动添加翻译
- **填写**: 确保使用你指定的翻译

建议对于重要术语填写明确的翻译，次要术语可以留空让 AI 自动填充。
