# 🏠 书籍库与主页（Library & Home）

主页用于总览项目状态并快速进入常用流程。系统数据默认存储在本地 **IndexedDB**。

---

## 📊 统计概览

主页统计区会显示：

- 📚 **总书籍**：当前书籍总数
- 📑 **总章节**：全部书籍章节总数
- 📝 **总字数**：基于章节内容统计的总字数（异步更新）
- ⭐ **收藏**：已收藏书籍数量

---

## 🚀 快捷操作

主页提供 4 个快捷入口：

### 1. ➕ 添加书籍

手动创建新书籍。

- **基础信息**：标题、作者、简介、标签、封面。
- **可选配置**：别名、Web URL、书籍级特殊指令（翻译/润色/校对）。

### 2. 🌐 从网站导入

通过链接抓取小说信息与章节内容。

- **支持站点**：`ncode.syosetu.com`、`novel18.syosetu.com`、`kakuyomu.jp`、`syosetu.org`。
- **章节勾选导入**：可按需选择章节。
- **重复导入识别**：会提示未导入/可能更新的章节，便于增量抓取。

### 3. 📚 查看所有书籍

跳转到书籍列表页，进行完整管理：

- **搜索过滤**：按书名、别名、作者、简介、标签筛选。
- **多维排序**：创建时间、更新时间、标题、章节数、字数、收藏优先等。
- **常用管理**：编辑、收藏/取消收藏、删除。
- **导入书籍**：支持从 JSON 导入（也支持从“导入资料”导入完整数据）。

### 4. 🤖 AI 设置

跳转到 AI 模型管理页面，配置可用模型与任务默认模型。

---

## ⌚ 最近编辑

主页会展示最近编辑的 **6 本书籍**（按 `lastEdited` 倒序）：

- 当书籍被更新后，会自动上浮到前列。
- 点击卡片可进入书籍详情页。

---

## 🔒 数据备份与安全

Tsukuyomi 默认本地存储，支持手动备份与云端同步。

- **手动备份**：建议定期在“设置 -> 导入/导出”执行导出资料。
- **Gist 同步**：可在设置中配置 GitHub Gist 做跨设备同步。

---

## ❓ 常见问题 (FAQ)

**Q: 网站导入失败怎么办？**
A: 先检查 URL 与网络连通性；在 Web 环境可在“设置 -> 代理设置”调整代理参数。

**Q: 可以一次性导出全部数据吗？**
A: 可以。在“设置 -> 导入/导出”使用“导出资料”。

**Q: 字数统计为什么不更新？**
A: 字数统计是后台异步计算的。如果您刚导入大量章节，可能需要稍等片刻待计算完成后才会刷新显示。
