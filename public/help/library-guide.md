# 🏠 主页 (Home) 使用指南

主页是 Tsukuyomi 的主页，您可以在这里管理所有书籍项目、查看统计数据、快速访问最近编辑的书籍。

---

## 📊 统计概览

页面顶部展示了您的翻译工作的整体数据：

- 📚 **总书籍**: 您当前管理的所有书籍数量
- 📑 **总章节**: 所有书籍的累计章节数
- 📝 **总字数**: 所有章节的累计字符数（自动统计）
- ⭐ **收藏**: 您标记为收藏（星标）的书籍数量

> 💡 **提示**: 字数统计是异步计算的，首次加载书架时可能会看到加载动画。

---

## 🚀 快速操作

书架提供了四个核心快捷操作：

### 1. ➕ 添加书籍

**用途**: 手动创建新书籍项目

**操作流程**:

1. 点击 **添加书籍** ➕ 按钮
2. 填写书籍基本信息：
   - 📖 **标题** ✱ (必填): 书籍的主标题
   - 🏷️ **别名** (可选): 其他译名或简称
   - 👤 **作者** (可选): 原作者名称
   - 📄 **简介** (可选): 书籍简介或剧情梗概
   - 🏷️ **标签** (可选): 用于分类的标签（如"轻小说"、"奇幻"）
   - 🖼️ **封面** (可选): 上传或粘贴封面图片
3. 点击 **保存** 💾 完成创建

**🔧 高级选项**:

- 🌐 **翻译指令**: 针对该书籍的特殊翻译要求（如"保留日式敬语"）
- ✨ **润色指令**: 润色任务的额外说明
- ✓ **校对指令**: 校对任务的特殊要求

### 2. 🌐 从网站导入

**用途**: 自动从支持的小说网站抓取书籍元数据和章节内容

**支持的网站**:

- 📖 **小説家になろう (Shousetsuka ni Narou)** - `syosetu.com`
- 📚 **カクヨム (Kakuyomu)** - `kakuyomu.jp`
- 🌸 **ハーメルン (Hameln)** - `syosetu.org`

**操作流程**:

1. 📋 复制小说的 URL（通常是主页链接）
2. 点击 **从网站导入** 🌐 按钮
3. 粘贴 URL 并点击 **获取信息** 🔍
4. 🤖 系统会自动抓取：
   - 📖 书名、作者、简介
   - 🖼️ 封面图片
   - 📑 章节列表和内容
5. 确认信息无误后点击 **导入** ⬇️

> ⚠️ **注意**: 网络抓取需要时间，请耐心等待。如果抓取失败，请检查 URL 是否正确或网站是否可访问。

### 3. 📚 查看所有书籍

**用途**: 跳转到书籍列表页，浏览完整的书架

**功能亮点**:

- 🔍 支持按标题、作者、标签搜索
- 📊 多种排序方式（最近编辑、创建时间、标题）
- ⭐ 筛选收藏书籍
- ⚡ 批量操作（导出、删除）

### 4. 🤖 AI 设置

**用途**: 快速跳转到 AI 模型配置页

**应用场景**:

- ➕ 添加新的 AI 模型
- ⚙️ 调整默认模型参数
- 🔄 切换翻译/润色任务的默认模型

## 📚 最近编辑

展示最近修改过的 **6 本书籍**，按编辑时间倒序排列。

### 书籍卡片信息

每张卡片显示：

- **封面**: 自动生成或您自定义的封面
- **标题**: 书名（最多显示 2 行）
- **作者**: 原作者名称
- **章节数**: 该书的总章节数
- **字数**: 该书的总字符数
- **更新时间**: 距离上次编辑的时间（如"今天"、"3天前"）

### 交互

- **点击书籍卡片**: 跳转到书籍详情页
- **悬停**: 卡片会有缩放和高亮效果

## 💡 使用技巧

### 快速开始翻译

1. **首次使用**:
   - 先配置 AI 模型（点击 **AI 设置**）
   - 添加或导入第一本书
   - 进入书籍详情页开始翻译

2. **日常工作流**:
   - 从书架点击最近编辑的书籍，直接继续翻译
   - 使用 **从网站导入** 批量导入新章节

### 封面管理

- **自动封面**: 如果未设置封面，系统会根据书名自动生成一个默认封面
- **自定义封面**: 编辑书籍时，可以上传本地图片或粘贴图片 URL
- **封面历史**: 系统会记录您使用过的封面，方便复用

### 数据备份

虽然数据会自动保存到本地浏览器，但强烈建议：

- 定期使用 **书籍列表页** 的导出功能
- 配置 **Github Gist 同步**（在设置中）实现云端备份

## ❓ 常见问题

**Q: 书架上的字数统计不准确怎么办？**
A: 字数是异步计算的，刷新页面后会重新统计。如果仍然不对，可能是章节数据需要重新加载。

**Q: 可以隐藏某些书籍吗？**
A: 目前不支持隐藏功能，但您可以：

- 删除不需要的书籍
- 使用标签和搜索功能快速定位目标书籍

**Q: 从网站导入的书籍能否批量更新章节？**
A: 可以。在书籍详情页，使用 **批量抓取** 功能可以检测并导入新章节。

**Q: "最近编辑" 的排序依据是什么？**
A: 排序依据是 `lastEdited` 时间戳，每次修改书籍信息、添加章节、翻译内容时都会自动更新。
